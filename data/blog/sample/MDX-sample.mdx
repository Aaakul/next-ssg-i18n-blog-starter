### Built-in image component

The component `./components/Image.tsx` provides built-in support for prioritized WebP image loading with automatic fallback, enabling more efficient image loading in modern browsers. During the loading process, SVG placeholders are used to enhance the user experience. When starting the development server or running production builds, the `genwebp` script automatically generates WebP copies of local images in PNG and JPG formats, using Node.js's `worker_threads` multithreading mechanism.

For local article banner images in PNG and JPG formats, you do not need to manually set the path for the WebP version.

## Github-style Alert

```md
> [!NOTE]
> Useful information that users should know, even when skimming content.

> [!TIP]
> Helpful advice for doing things better or more easily.

> [!IMPORTANT]
> Key information users need to know to achieve their goal.

> [!WARNING]
> Urgent info that needs immediate user attention to avoid problems.

> [!CAUTION]
> Advises about risks or negative outcomes of certain actions.
```
